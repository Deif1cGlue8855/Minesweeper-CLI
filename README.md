If you want to tweak any of the code, it has to be compiled with the ncurses library

For Nvim:

!g++ % -lncurses -o mineSweeper


# Conrols:

h - left
l - right
k - up
j - down

a - flag
s - select tile

r - restart
q - change difficulty
c - change board size (only works on custom difficulty)

# Info

Info such as how many games played, wins, losses and win streaks are stored for each gamemode
